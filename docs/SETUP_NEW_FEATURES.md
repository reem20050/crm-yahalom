# הוראות התקנה לתכונות החדשות

## מה נוסף למערכת:

1. **ניהול משמרות** - דף `/shifts` לניהול משמרות
2. **ניהול משימות** - דף `/tasks` לניהול משימות
3. **לוח שנה** - דף `/calendar` לתצוגה ויזואלית
4. **דוחות** - דף `/reports` עם גרפים וסטטיסטיקות
5. **התראות** - פעמון התראות בנאבבר

## שלבי התקנה:

### 1. התקנת Dependencies חדשות (Frontend)

```bash
cd frontend
npm.cmd install
```

זה יתקין:
- `react-big-calendar` - ללוח שנה
- `recharts` - לגרפים
- `date-fns` - לעבודה עם תאריכים

### 2. הפעלת השרתים מחדש

**Backend:**
```bash
cd backend
START_SERVER.bat
```

**Frontend:**
```bash
cd frontend
START_FRONTEND.bat
```

### 3. רענון הדפדפן

לחץ `Ctrl + F5` (או `Cmd + Shift + R` ב-Mac) לרענון מלא.

### 4. בדיקת הקישורים החדשים

בנאבבר אמור להופיע:
- 📊 לוח בקרה
- 🕒 משמרות
- ✅ משימות
- 📅 לוח שנה
- 📈 דוחות
- 👥 עובדים
- 🏢 לקוחות
- 🔔 התראות (פעמון)

## אם יש שגיאות:

1. פתח את הקונסול בדפדפן (F12)
2. בדוק אם יש שגיאות אדומות
3. אם יש שגיאה על `react-big-calendar` או `recharts` - צריך להריץ `npm.cmd install` שוב

## הערות:

- כל התכונות תומכות בעברית מלאה (RTL)
- התראות מתעדכנות אוטומטית כל 30 שניות
- לוח שנה מציג משמרות ומשימות
- דוחות כוללים גרפים ויזואליים
