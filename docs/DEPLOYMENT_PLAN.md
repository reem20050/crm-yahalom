# תוכנית פריסה - מערכת CRM צוות יהלום

## סקירה כללית
תוכנית זו מכסה 3 אפשרויות להריץ את המערכת תמיד:
1. **הפעלה אוטומטית מקומית** (Windows Service/Task Scheduler)
2. **דומיין חינמי עם Tunnel** (ngrok/localhost.run/Cloudflare Tunnel)
3. **העלאה ל-Cloud חינמי** (Railway/Render/Vercel/Netlify)

---

## אפשרות 1: הפעלה אוטומטית מקומית

### א. הפעלה ידנית (פשוט)
- הרץ `START_ALL.bat` - מפעיל את שני השרתים בחלונות נפרדים

### ב. Windows Task Scheduler (הפעלה אוטומטית בהפעלה)
- השרתים יופעלו אוטומטית בכל הפעלה של Windows
- לא דורש הרשאות מנהל
- קל להגדרה

### ג. Windows Service (הפעלה תמידית ברקע)
- השרתים רצים כ-Service ברקע
- לא צריך להיות מחובר
- דורש NSSM (Non-Sucking Service Manager)

---

## אפשרות 2: דומיין חינמי עם Tunnel

### א. ngrok (מומלץ)
- **יתרונות:** יציב, HTTPS אוטומטי, דומיין קבוע בתשלום
- **חסרונות:** דורש הרשמה, דומיין משתנה בגרסה חינמית
- **מחיר:** חינמי (דומיין משתנה) / $8/חודש (דומיין קבוע)

### ב. localhost.run (חינמי לחלוטין)
- **יתרונות:** חינמי, לא דורש הרשמה
- **חסרונות:** דומיין משתנה בכל הפעלה, פחות יציב
- **מחיר:** חינמי

### ג. Cloudflare Tunnel (cloudflared)
- **יתרונות:** חינמי, יציב, HTTPS אוטומטי
- **חסרונות:** דורש הרשמה ל-Cloudflare
- **מחיר:** חינמי

### ד. localhost.run (SSH Tunnel)
- **יתרונות:** חינמי, פשוט
- **חסרונות:** דורש SSH, דומיין משתנה
- **מחיר:** חינמי

---

## אפשרות 3: Cloud Hosting חינמי

### א. Railway (מומלץ)
- **יתרונות:** חינמי $5/חודש, קל להגדרה, אוטומטי
- **חסרונות:** דורש כרטיס אשראי (לא מחויב)
- **מחיר:** $5 חינמי/חודש

### ב. Render
- **יתרונות:** חינמי, אוטומטי
- **חסרונות:** שירותים "נרדמים" אחרי 15 דקות ללא שימוש
- **מחיר:** חינמי (עם הגבלות)

### ג. Vercel (לפרונטאנד) + Render (לבקאנד)
- **יתרונות:** Vercel מעולה לפרונטאנד, Render לבקאנד
- **חסרונות:** צריך 2 שירותים נפרדים
- **מחיר:** חינמי

### ד. Netlify (לפרונטאנד) + Railway (לבקאנד)
- **יתרונות:** Netlify מעולה לפרונטאנד
- **חסרונות:** צריך 2 שירותים נפרדים
- **מחיר:** חינמי
