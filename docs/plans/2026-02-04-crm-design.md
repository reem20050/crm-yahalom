# מערכת CRM - צוות יהלום
## תאריך: 2026-02-04

## סקירה כללית
מערכת CRM מקיפה לחברת אבטחה "צוות יהלום" - חברה בינונית (20-100 עובדים) המספקת שירותי שמירה ואבטחה פיזית לעסקים ואירועים.

## מודולים

### 1. דשבורד ראשי
- כרטיסיות סיכום: לידים, עסקאות, הכנסות, משמרות
- פעילות מכירות: לידים ממתינים, הצעות מחיר, חוזים לחידוש
- תפעול יומי: משמרות היום, התראות, אירועים קרובים
- גרף הכנסות והתראות כספיות

### 2. ניהול לידים ומכירות
- מקורות: אתר, וואטסאפ, טלפון, המלצות
- משפך: ליד חדש → פגישה → הצעת מחיר → משא ומתן → סגירה
- התראות מעקב אוטומטיות

### 3. ניהול לקוחות
- פרטי חברה ואנשי קשר
- חוזים ותנאי התקשרות
- אתרים/נקודות שמירה
- היסטוריית פעילות

### 4. ניהול עובדים ומשמרות
- כרטיס עובד: פרטים, מסמכים, כישורים
- לוח משמרות שבועי/חודשי
- שיבוץ חכם
- דיווח נוכחות מהטלפון

### 5. ניהול אירועים
- אירועים חד-פעמיים
- תכנון דרישות אבטחה
- שיבוץ צוות
- מעקב סטטוס

### 6. דוחות וכספים
- דשבורד כספי
- מעקב תשלומים
- דוחות: מכירות, לקוחות, עובדים, אירועים

## אינטגרציות
- Google Workspace (Gmail, Calendar, Drive)
- WhatsApp Business API
- חשבונית ירוקה API

## טכנולוגיה
- Frontend: React + TypeScript + Tailwind CSS
- Backend: Node.js + Express
- Database: PostgreSQL
- Authentication: JWT

## משתמשים
- מנהל ראשי - גישה מלאה
- מנהלי משמרות - שיבוצים ועובדים
- אנשי מכירות - לידים ולקוחות
- עובדים - צפייה במשמרות
